# SOTE -ProjectHub API

> Backend API for the ProjectHub application which is a Project collection system

## Usage

Rename "config/config.env.env" to "config/config.env" and then update the values/settings to your own

## Install Dependencies

```
npm install
```

# Run App

```
# Run in dev mode
    npm run dev

#Run in prod mode
    npm start
```
